---
title: "Patient Invoicing with Payments"
api: "POST https://api.useforesee.com/createInvoice"
description: "This endpoint creates an easy to understand patient invoice and provides a checkout page for completing the payments"
---

### Body

<ParamField body="From" type="string">
  This denotes the entity who wants to send an invoice to their patients
</ParamField>

<ParamField body="From-Address" type="string">
  This denotes the address of the entity who wants to send an invoice to their patients
</ParamField>

<ParamField body="Patient-Name" type="string">
  The name of the patiet to whom the invoice is addressed to
</ParamField>

<ParamField body="Amount(Optional)" type="string">
  The amount due to be collected. If Patient Name is given, the API will 
  automatically calculate the pending balance from your EHR and auto populate the amount 
</ParamField>

### Response

<ResponseField name="success" type="number">
  if successful returns 200
</ResponseField>



<RequestExample>

```bash Example Request
curl --location --request POST 'https://api.useforesee.com/createInvoice' \
--header 'Content-Type: application/json' \
--header 'Authorization: Token <token>' \
--data-raw '{
    "From": "Camino Family Dentistry",
    "From-Address": "451 W El Camino Real",
    "Patient-Name": "Vivek Anand Murthy",
    "Amount": "0.0"
}'
```

</RequestExample>

<ResponseExample>

```json Response
"https://dev.payment.useforesee.com/invoice/70709a8f-2ec0-4310-9d22-83db4d52dcdb"

```

</ResponseExample>
